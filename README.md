# Project Overview
This project is about being a Student Digital Ambassador. The goal is to help friends understand and use the world well. The project is about learning digital skills. These skills include making an online profile using the internet safely understanding digital communication and knowing about cyber threats.
The project has five parts. Each part helps learn a digital skill. All tasks are in this GitHub repository. There are also supporting files, pictures and a detailed report. Through this project I learned to use platforms and understand professional digital behavior, online safety and using technology ethically.

# Module Summaries

* Module 1: Digital Literacy Infographic

In this part I made an infographic. It explains literacy and its importance. The infographic talks about internet use responsible social media use, digital footprints and cybersecurity. I made sure the information is clear and easy to understand. This activity helped me be more creative and design things well.



* Module 2: Digital Portfolio

In this part I made my online profile using GitHub and LinkedIn. I added my details, achievements, skills and projects. I learned to make my profile attractive and informative. This part helped me understand the importance of having an online presence. It can help with job opportunities.

* Module 3: Coding & Collaboration Platforms

In this part I used coding and collaboration platforms like HackerRank and Google Forms. I practiced solving programming problems. I also made a literacy quiz using Google Forms. This part helped me understand tools and online assessments. It showed me the importance of learning skills in technology.

* Module 4: Email Etiquette & Communication

In this part I learned about communication online. I wrote emails for academic and professional purposes. I also made a checklist of social media dos and don’ts. This part helped me improve my communication skills. It taught me to present myself online.

* Module 5: Cybercrime Awareness

In this part I studied cybercrime cases. I learned about cyber threats like phishing and hacking. I made a prevention checklist. It has practices, for staying safe online. This part helped me understand cybersecurity concepts. It taught me to be cautious and responsible online.
